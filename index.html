Sierpinski Triangle

The Sierpinski triangle is a fractal described in 1915 by Waclaw Sierpinski. It is a self similar structure that occurs at different levels of iterations, or magnifications.

My Implementation relies on the following strategy:

  1. Draw base triangle
  2. Draw inverted triangle inside base triangle
  3. Repeat

Inscribing an inverted triangle inside the base triangle creates more triangles which themselves can be inscribed. This process can be repeated infinitely. My approach uses Pythagoras Theorem to derive the necessary X,Y coordinates for the base and inverted triangles. Once you know the coordinates for the base triangle and the inscribed triangle, the process can be repeated recursively until told to stop.
